# Recruitment Data Analysis Dashboard

## Project Overview

This project focuses on analyzing recruitment pipeline performance using Python and Power BI. The objective is to track candidate progress through various hiring stages and identify trends that help improve recruitment efficiency.

---

## Tools Used
* Python (Pandas, NumPy)
* Google Colab
* Power BI
* GitHub

---

## Data Cleaning Process
The raw recruitment dataset was cleaned using Python (Pandas).

### Cleaning Steps Performed
* Removed duplicate records
* Checked and handled missing values
* Standardized Status Column
* Corrected date formats
* Fixed inconsistent text values
* Converted data types
* Validated final dataset for reporting

---

## Key Performance Indicators (KPIs)
* Total Submissions
* Total Interviews
* Total Offers
* Total Joins
* Interview Rate%
* Candidate Conversion Rate %

---

## Dashboard Features

### Recruitment Funnel Analysis

Tracks candidate movement through:

Submission → Interview → Offer → Join

### Trend Analysis

* Monthly submission trends
* Hiring performance over time

### Recruiter Performance

* Recruiter-wise submissions
* Interviews
* Offers
* Joins
* Join rate %

### Client Analysis

* Submission distribution by client

### Job Role Analysis

* Submission volume by job role

### Location Analysis

* Submission distribution by location

---

## Key Insights

* Identified top-performing recruiters based on join rate.
* Analyzed monthly hiring trends.
* Evaluated client-wise recruitment contribution.
* Measured overall candidate conversion efficiency.
* Compared job role demand and location-wise hiring activity.
